📌 Problem Statement

Spotify’s raw “Top 50” dataset primarily provides ranking lists and numerical metrics without consolidated analytical insights. This creates several business limitations:
No centralized KPI monitoring → Stakeholders cannot quickly view total songs, artists, average popularity, or duration metrics in one place.
Lack of content type visibility → No structured comparison between explicit and non-explicit tracks.
Limited release format insights → No breakdown of singles vs albums vs compilations.
No trend tracking → Monthly and yearly patterns in song releases and popularity are difficult to identify.
Artist and song performance disconnected → No drill-down capability linking high-level insights to detailed track-level records.
No volatility analysis → Difficult to identify stable hits versus fluctuating rankings.
Decision-making gaps → Marketing and playlist teams lack structured insights to identify which artists, formats, or song lengths perform best.

The developed dashboard solves these issues by integrating KPIs, trend analysis, distribution breakdowns, artist-level insights, and ranking volatility analysis into a single interactive analytics platform.
